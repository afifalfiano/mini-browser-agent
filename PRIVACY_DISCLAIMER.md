# Privacy Disclaimer

**Important Notice Regarding Privacy and Security**

By using the **MiniMax Browser Agent** extension, you acknowledge and agree to the following terms regarding your privacy and data security. 

### 1. Local Data Storage
All data related to the extension's internal operations—including your API keys, conversation history, saved sessions, and extracted memory facts—are stored **locally** on your device using Chrome's local storage (`chrome.storage.local`). We do not host, operate, or maintain any central database or telemetry servers that collect your personal data.

### 2. Third-Party API Data Transmission
To function as an AI agent, this extension must send data to third-party AI providers (e.g., MiniMax, Google Gemini) based on your chosen configuration. 
When you issue a command or ask a question, the extension may capture and transmit:
- Your prompt/chat messages.
- The visible text and DOM structure of the web page you are currently viewing.
- Screenshots of the active tab.
- Metadata such as URLs and page titles.

### 3. Out of Scope Responsibility
**We have no control over how third-party API providers handle, store, use, or process the data sent to them.** 
Any privacy concerns, data breaches, or data usage policies related to the AI models are strictly governed by the respective providers. 
- Please review the [MiniMax Privacy Policy](https://www.minimaxi.chat) before using their API.
- Please review the [Google Gemini Privacy Policy](https://ai.google.dev/) before using their API.

### 4. Sensitive Information Warning
Because the AI can "read" the page you are on, **DO NOT** use this extension on web pages that contain highly sensitive, personal, financial, medical, or confidential information. If the AI is triggered on such pages, that sensitive content may be transmitted to the third-party API provider as context.

### 5. "As-Is" Software
This extension is provided "as is" without any warranties of any kind. The developers and contributors of this extension are not responsible for any privacy leaks, data loss, account bans, or unauthorized access resulting from the use of this extension or the third-party APIs it connects to.
