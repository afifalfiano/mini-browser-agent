Privacy & Security

Your data belongs to you. Period.

🔒 Local Only

All sessions, history, and memory facts are stored locally in your Chrome storage. We don't have a backend.

📡 Direct API

The extension communicates directly with your chosen AI provider. No middleman servers involved.

🛠️ Open Audit

Being 100% open source means anyone can audit the code to ensure data integrity and security.

Common Questions

Do I need an API Key?

Yes, you need to provide your own MiniMax API key to use the extension.

Is it free to use?

The extension is free and open-source. However, you are responsible for any costs incurred from the AI provider API usage.

Does it work on any website?

It works on most standard websites. However, extremely complex SPAs or sites with heavy anti-bot protections might limit the agent's actions.

Ready to Upgrade your Browsing?

Install the unpacked extension in Developer Mode and start automating today.

  1. Clone the repository from GitHub
  2. Open chrome://extensions in Chrome
  3. Enable Developer mode
  4. Click Load unpacked and select the project folder
