Four Powerful Modes

Designed specifically to handle various types of web scenarios.

🤖

Agent Mode

The default mode. Fully autonomous multi-step execution. Navigates, clicks, and types without needing step-by-step guidance.

📖

Reader Mode

Focuses purely on text and data. Summarizes long articles and extracts specific information without altering the page.

📝

Form Filler

A data entry specialist. Analyzes the form structure, fills out fields step-by-step, and reviews data before submission.

🔬

Researcher

Designed for cross-site research. Opens new tabs, compares information from multiple sources, and provides source links.
