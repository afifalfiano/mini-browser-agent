# Landing Page Design System

## Color Palette
- **Background:** `#050508` (Deep Space Black)
- **Primary:** `#6366f1` (Indigo Glow)
- **Secondary:** `#a855f7` (Purple Pulse)
- **Accent:** `#0ea5e9` (Sky Blue)
- **Text Primary:** `#ffffff` (Pure White)
- **Text Secondary:** `#94a3b8` (Slate Gray)
- **Gradients:** `linear-gradient(135deg, #6366f1 0%, #a855f7 100%)`

## Typography
- **Headings:** `Outfit`, sans-serif (800 weight for Hero, 700 for Sections)
- **Body:** `Inter`, sans-serif (400 for text, 500 for UI elements)

## Components
- **Buttons:** 
  - Primary: Gradient background with subtle outer glow.
  - Secondary: Ghost style with glassmorphism border.
- **Cards:** Glassmorphism (`rgba(255,255,255,0.03)`) with 1px border (`rgba(255,255,255,0.1)`).
- **Hero Image:** Should be flat and clear, no extra container transforms if the image is already 3D.
- **Features Showcase:** High-impact grid with large icons and clear separation.

## Animations
- **Fade-In:** Upward motion for all sections.
- **Hover:** Subtle scaling and border-color transition.
